# MIS-503

All of these files are Word documents of an R-Markdown document. The files in this repository are as follows

## Candy Analysis

Visualize candy production using a line chart, scatterplot, and bar chart.


## DiamondsandPrices

Analyze and visualize various aspects of diamonds, including size and price. From the infamous “Diamonds” dataset. 

## MigrationPatterns

After cleaning the data, analyze and visualize world migration patterns by continent. 

## StatAnalysis

Analyze and visualize BMI data using t-tests, linear regression, and ANOVA. 

## WakeCountyHomeValues

Using data from Zillow, analyze and visualize home sales and rentals by city and year, and compare with other regions. This was our final project.
